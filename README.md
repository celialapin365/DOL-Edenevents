## **找不到跳转的剧情小节怎么办？**

1.通常来说不会离太远

2.在目录中寻找跳转

3.利用word自带的查找功能（开始-编辑-查找）

4.没有文本的小节（多半是以Sex或者Molestation结尾的纯遭遇战）不做整理，不过多半会有Sex/Molestation Finish小节，作为遭遇战的结果

5.我遗漏了，请给我回复留言！
